整理工作生活中不错的软件和插件。

软件

  • Bandizip 轻量级、快速且整合程度较高的解压缩软件
  • Chrome 快速、轻松且安全地使用网络的浏览器
  • Cmder 美观、强大的终端工具
  • Git 分布式版本控制器客户端
  • IntelliJ IDEA 智能、强大的Java IDE
  • Navicat 强大的跨平台数据库管理和设计工具
  • Postman 强大好用的API开发工具
  • Potplayer 播放器
  • Pycharm 智能、强大的Python IDE，甚至可以教你学Python
  • RedisDesktopManager Redis可视化工具
  • Robo 3T MongoDB可视化工具
  • Snipaste 截图+贴图的好工具
  • Teamviewer 远程控制软件
  • TortoiseSVN Subversion版本控制系统客户端
  • Typora Markdown编辑，复制表格内容甚至可以帮你自动生成md语法
  • Visual Studio Code 不仅仅是编辑器
  • WinRAR 一直还生存的老牌压缩软件
  • Xmind 思维导图
  • Xshell 强大的安全终端模拟软件
  • Xftp sftp和ftp文件传输客户端
  • 向日葵 远程控制软件
